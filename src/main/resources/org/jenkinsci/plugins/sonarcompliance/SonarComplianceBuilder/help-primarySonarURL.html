The URL from where the sonar metric key values to be obtained.

If its not differential, then absolute sonar metric values are obtained and substituted as values for the metric keys in rules.

If its differential and no secondary sonar url, then the difference of metric key values of 2 successive runs are substituted as values for the metric keys in rules.

If its differential and secondary sonar url is provided, then the difference of metric key values of primary and secondary sonar are substituted as values for the metric keys in rules.
